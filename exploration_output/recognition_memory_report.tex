\section{Recognition Memory Analyses}

\subsection{Normality Assessment}
Before conducting comparative statistical tests for all three hypotheses, participant-level mean accuracy and response times were assessed for normality using the Shapiro-Wilk test. All metrics significantly violated the assumption of normality:

\begin{itemize}
    \item \textbf{H1: Overall Accuracy}
        \begin{itemize}
            \item \textbf{Abrupt Cut (AB):} Statistic = 0.9567, $p = 0.0091$ (NOT Normal)
            \item \textbf{Natural Cut (NB):} Statistic = 0.9419, $p = 0.0007$ (NOT Normal)
        \end{itemize}
    \item \textbf{H1: Response Times}
        \begin{itemize}
            \item \textbf{Abrupt Cut (AB):} Statistic = 0.8817, $p < 0.0000$ (NOT Normal)
            \item \textbf{Natural Cut (NB):} Statistic = 0.9512, $p = 0.0025$ (NOT Normal)
        \end{itemize}
    \item \textbf{H2: Before Boundary (BB) Frames}
        \begin{itemize}
            \item \textbf{AB:} Statistic = 0.9475, $p = 0.0027$ (NOT Normal)
            \item \textbf{NB:} Statistic = 0.9218, $p = 0.0001$ (NOT Normal)
        \end{itemize}
    \item \textbf{H3: Event-Middle (EM) Frames}
        \begin{itemize}
            \item \textbf{AB:} Statistic = 0.9226, $p = 0.0001$ (NOT Normal)
            \item \textbf{NB:} Statistic = 0.9183, $p = 0.0000$ (NOT Normal)
        \end{itemize}
\end{itemize}

\emph{See Figures~\ref{fig:h1_qq}, \ref{fig:h2_qq}, and \ref{fig:h3_qq} for Q-Q plots.}
